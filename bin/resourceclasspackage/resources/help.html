How to use unitED

When you open the program the calculator should look like this:

To start calculating, click a number or select a unit from the drop down menu:

Next select an operator to calculate with, keep in mind you can only add and subtract same type units but can multiply and divide for conjunctions:

You can now calculate by pressing the equals button:

After doing that you can either click an operator to continue calculating with your previous answer or click a number to start a new calculation

You can also add custom units for an of your calculating needs. Just click on the custom unit option from the dropdown menu and it will prompt you for a new unit!

To find out more about us you can navigate to the Help menu on the menu bar and click the About option:

To reopen this help page you can do something similar but select the Help option:

And finally to close the program you can go to either the top right corner and press the red x or go to the file button in the top left and then select the exit option
